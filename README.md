# 🎯 Establecimiento de Metas - Método Fénix

Una herramienta web minimalista e interactiva diseñada para la planificación estratégica personal, basada en los principios de **Brian Tracy**. Permite a los usuarios definir sus objetivos con claridad y exportar un plan de acción detallado en formato **PDF**.

## 🚀 Demo en Vivo
Puedes probar la aplicación funcionando aquí:
👉 **[VER DEMO DEL PROYECTO](https://debescreerenti23.github.io/make-gol/)** 

## ✨ Características
* **Interfaz Cyberpunk/Minimalista:** Diseño oscuro con gradientes neón y efectos de cristal (glassmorphism).
* **Efecto Glow Pulse:** Título principal con animación de parpadeo suave mediante CSS.
* **Cuestionario Estratégico:** 6 preguntas clave para desglosar metas, obstáculos y habilidades necesarias.
* **Generación de PDF:** Exportación inmediata de tus respuestas en un documento formateado profesionalmente.
* **Diseño Responsivo:** Optimizado para escritorio y dispositivos móviles.

## 🛠️ Tecnologías utilizadas
* **HTML5** - Estructura semántica.
* **CSS3** - Animaciones personalizadas y estilos de diseño.
* **Tailwind CSS** - Framework para el diseño visual rápido y moderno.
* **JavaScript (ES6+)** - Lógica de manejo de fechas y manipulación del DOM.
* **jsPDF** - Librería para la generación dinámica de documentos PDF en el cliente.
